# 🚀 Scanlation Toolkit V2 - All-in-One Studio

Scanlation (Manhwa/Manga) süreçlerinizi tarayıcı üzerinden, harici programlara (Photoshop, vb.) ihtiyaç duymadan yönetmenizi sağlayan **hepsi bir arada** çalışma istasyonu.

## 🛠️ Temel Özellikler

* **🎨 Renk Korumalı Levels Motoru:** Görüntüleri griye çevirmeden, RGB kanalları üzerinden dengeli temizlik (RAW Cleaning) yapar.
* **🧩 Modüler Engine Mimarisi:**
    * **Panel Birleştirici:** Düzensiz panelleri otomatik hizalar ve birleştirir.
    * **Dönüştürücü (Converter):** Gelişmiş kontrast ve seviye ayarları.
    * **Watermark Motoru:** PNG logolarınızı sayfaların üzerine toplu şekilde işler.
* **💾 Kalıcı Oturum (Auto-Save):** Tarayıcıyı kapatsanız bile çalışma havuzunuz `localStorage` sayesinde korunur.
* **⚡ Hızlı İş Akışı:**
    * **Ctrl+Z (Geri Al):** Hatalı işlemleri anında geri al.
    * **Klavye Kısayolları:** `g` (Grup), `z` (ZIP), `c` (Sıfırla) ve yön tuşları ile mouse kullanmadan çalış.
* **🖥️ Modern Arayüz:** JetBrains Mono fontu ve Tailwind CSS destekli göz yormayan karanlık mod.

## ⌨️ Klavye Kısayolları

| Kısayol | İşlev |
| :--- | :--- |
| **G** | Seçili panellerden grup oluştur |
| **Z** | ZIP olarak dışa aktar |
| **C** | Tüm ayarları sıfırla |
| **Boşluk** | Sayfa seç / Seçimi kaldır |
| **Sağ/Sol Ok** | Sayfalar arası geçiş |

## 📜 Sürüm Notları (V2.0.0)

Bu sürüm, **V11** temelinin üzerine inşa edilmiş tamamen modüler bir "Studio Edition"dır. Eski sistemin sınırlamaları kaldırılmış, performans ve veri güvenliği odaklı mimariye geçilmiştir. 
- **Geliştirmeler:** RGB bazlı renk korumalı işleme, yerel veritabanı (localStorage) desteği, geri al (undo) fonksiyonu ve tam modüler arayüz.

## 💻 Geliştirme

Bu proje saf **HTML, JavaScript (ES6+) ve Tailwind CSS** kullanılarak geliştirilmiştir. Herhangi bir sunucu tarafı bağımlılığı yoktur, tüm işlemler tarayıcınızda yerel (client-side) olarak gerçekleşir.
